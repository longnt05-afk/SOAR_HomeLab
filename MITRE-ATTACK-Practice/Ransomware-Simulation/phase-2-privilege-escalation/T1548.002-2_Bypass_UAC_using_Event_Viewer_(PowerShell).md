**T1548.002-2 ****Bypass**** UAC ****using**** ****Event**** ****Viewer**** (****PowerShell****)**
**1. Executive Summary**
Trong phiên kiểm thử SOC/SOAR Home Lab, nhóm thực hiện mô phỏng kỹ thuật MITRE ATT&CK T1548.002 - Abuse Elevation Control Mechanism: Bypass User Account Control bằng Atomic Red Team.
Bài test mô phỏng hành vi lợi dụng cơ chế xử lý file .msc của Windows thông qua registry path:
**HKCU\Software\Classes\****mscfile****\shell\open\command**
Payload PowerShell tạo registry hijack để chuyển handler mở .msc sang cmd.exe, sau đó gọi eventvwr.msc. Khi Event Viewer/MMC xử lý file .msc, hành vi này có thể làm phát sinh command shell trong ngữ cảnh elevated nếu điều kiện UAC và quyền người dùng phù hợp.
Kết luận điều tra: Đây là True Positive - Authorized Simulation.

Cơ chế (map ATT&CK T1548.002 – Bypass UAC):
.msc file được Windows resolve qua class mscfile. Windows ưu tiên đọc HKEY_CURRENT_USER\Software\Classes trước khi fallback về HKEY_CLASSES_ROOT - đây là điểm attacker khai thác.
Script ghi đè handler open của mscfile trỏ sang cmd.exe thay vì mmc.exe mặc định.
eventvwr.msc (thực chất là eventvwr.exe) có manifest autoElevate=true - Windows tự nâng quyền binary này lên High Integrity mà không hỏi UAC, vì đây là binary Microsoft ký số, nằm trong thư mục protected.
Khi eventvwr.exe khởi động, nó internally trigger một lệnh liên quan tới association mscfile - do đã chạy ở quyền elevated, tiến trình con sinh ra (bị hijack thành cmd.exe) thừa hưởng luôn token elevated đó.
Kết quả: attacker chỉ cần user thường, không cần mật khẩu admin, vẫn có được cmd.exe chạy quyền cao - bypass UAC hoàn toàn im lặng.
